## Home budget application using React + Redux + Rails

### Description
Application which uses React + Redux + Rails.

It uses [*npm*](https://www.npmjs.com/) package manager.

### Usage
Setting up Rails:
```
$ bundle install
$ cp /config/database.yml.sample /config /database.yml
$ rake db:setup
```
Setting up npm:
```
$ npm install
```

Start rails server:
```
$ rails s
```
Start npm:
```
$ npm start
```
